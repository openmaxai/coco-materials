# Web 控制台 — 使用技巧

Web 控制台是基于浏览器的对话界面，无需安装任何应用，即可与 OpenMax AI 员工沟通。

## 访问 Web 控制台

设置 AI 员工时会提供你的 Web 控制台地址，格式通常如下：
```
https://[你的域名].icoco.ai/console/
```

可在桌面端或移动端的任意浏览器中访问。

## 基本使用

### 开始对话
1. 打开 Web 控制台地址
2. 在底部输入框中输入消息
3. 按 **Enter** 或点击**发送**

### 上传文件
点击**回形针图标**（📎）可附加：
- PDF 文档
- 图片（PNG、JPG、WEBP）
- 文本文件

### 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行（多行输入） |
| `↑` | 调出上一条消息 |
| `Esc` | 取消当前输入 |

## 嵌入到你的网站

可以将 Web 控制台作为聊天窗口嵌入网站：

### 方式 A：iFrame 嵌入
```html
<iframe
  src="https://[你的域名].icoco.ai/console/"
  width="400"
  height="600"
  frameborder="0"
  style="border-radius: 12px; box-shadow: 0 4px 24px rgba(0,0,0,0.1);"
></iframe>
```

### 方式 B：浮动挂件
在页面中引入 OpenMax 挂件脚本：
```html
<script>
  window.COCO_CONFIG = {
    botId: 'YOUR_BOT_ID',
    position: 'bottom-right',
    primaryColor: '#FFD646'
  };
</script>
<script src="https://icoco.ai/widget.js" async></script>
```

在 OpenMax Dashboard → Web 控制台 → 嵌入设置 中配置挂件参数。

## 高级功能

### 自定义欢迎语
设置用户首次打开控制台时显示的问候语：
> Dashboard → 机器人设置 → 欢迎消息

### 对话历史
- Web 控制台在浏览器会话中保存对话历史
- 用户可向上滚动查看完整对话
- 默认情况下，关闭浏览器标签页后历史记录清空

### 访问权限控制
- **开放访问**：任何拥有链接的人都可以对话（适合公开网站）
- **密码保护**：需要输入密码才能访问
- **单点登录（SSO）**：与现有认证系统集成（企业版）

## 进阶技巧

**复制回复**：将鼠标悬停在机器人的任意回复上，可以看到复制和点赞/踩按钮。

**重新生成**：如果想让机器人给出不同的回答，点击最后一条回复上的"重新生成"。

**分享对话**：使用分享按钮生成当前对话的链接。

**移动端**：Web 控制台完全适配移动端——无需安装任何应用，在手机和平板上均可正常使用。

**深色模式**：控制台自动跟随系统的深色/浅色模式设置。

## 故障排除

**控制台无法加载？**
- 检查订阅是否有效
- 清除浏览器缓存和 Cookie
- 尝试换用其他浏览器

**回复被截断？**
- 超长回复可能被截断——回复"继续"获取剩余内容

**文件上传失败？**
- 文件大小上限：20MB
- 支持格式：PDF、PNG、JPG、WEBP、TXT、MD、CSV
