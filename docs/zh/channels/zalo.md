# Zalo — 使用技巧

在 Zalo 上使用 OpenMax AI 员工。Zalo 是越南最受欢迎的即时通讯平台，拥有超过 7500 万用户。

## Zalo（官方）vs Zalo 个人版（非官方）

OpenMax 支持两种方式接入 Zalo，根据你的需求选择：

| | Zalo（官方） | Zalo 个人版（非官方） |
|---|---|---|
| **是什么** | 使用官方 [Zalo Bot Platform](https://bot.zaloplatforms.com) API 创建一个机器人，用户可以向其发送消息 | 使用真实 Zalo 账号通过扫码登录（类似 WhatsApp Web） |
| **协议** | 官方 API — 由 Zalo/VNG 官方支持 | 逆向工程协议（[zca-js](https://github.com/nicenathapong/zca-js)）— **未经 Zalo/VNG 官方认可** |
| **所需账号** | 一个 Zalo 账号用于在 Bot Platform 上注册机器人 | 一个专用 Zalo 账号（必须与对话用户的账号不同） |
| **设置方式** | 创建机器人 → 在 Dashboard 粘贴 Bot Token | 扫描二维码 — 无需 Token 或开发者设置 |
| **账号风险** | 无 — 使用官方 API | 可能被限制或封号 — 风险自担 |
| **部署时间** | 约 5 分钟 | 约 2 分钟 |
| **部署指南** | [Zalo（官方）部署指南 →](../getting-started/channel-deployment#zalo) | [Zalo 个人版（非官方）部署指南 →](../getting-started/channel-deployment#zalo-personal) |

| 功能 | Zalo（官方） | Zalo 个人版（非官方） |
|---|:---:|:---:|
| **文本消息**（收发） | ✅ | ✅ |
| **图片**（接收） | ✅ | ✅ |

| **文件与文档**（收发） | ❌ | ✅ |
| **语音消息**（发送） | ❌ | ✅ |
| **表情贴纸**（收发） | ✅ | ✅ |
| **贴纸搜索** | ❌ | ✅ |
| **表情回应**（收发） | ❌ | ✅ 6 种类型 |
| **引用回复** | ❌ | ✅ |

::: tip 我应该选择哪个？
**Zalo（官方）**稳定且无风险，但功能限于文本、图片和贴纸。如果稳定性比功能更重要，选择这个。

**Zalo 个人版（非官方）**支持完整的 Zalo 体验——文件、表情回应、语音、引用回复、智能群聊模式——但使用非官方协议，可能导致账号受限。如果你需要更丰富的功能并愿意接受这个风险，选择这个。
:::

## Zalo（官方）— 使用方法

### 私信
直接向你的 Zalo 机器人发送消息，无需特殊命令，机器人会立即回复。

### 开始使用
1. 在 Zalo 中搜索你的机器人名称
2. 开始对话 — 机器人自动回复
3. 第一个发送私信的用户将成为**管理员**（Owner）

### 权限管理
默认情况下，只有管理员可以与机器人对话。如需允许其他人使用：

**白名单模式** — 仅指定用户可以对话：
> 发送给机器人：`开启白名单模式，添加手机号 +84 xxx`

**开放模式** — 所有人均可对话：
> 发送给机器人：`开启开放模式，所有人都可以跟你聊天`

## Zalo 个人版（非官方）— 使用方法

### 私信
任何向你的专用机器人 Zalo 账号发送消息的人都会收到 AI 回复。

### 开始使用
1. 设置时 AI 员工会发送一个二维码给你
2. 使用专用机器人 Zalo 账号扫描
3. 连接建立 — 开始对话

### 权限管理
与 Zalo（官方）相同：默认仅管理员可用，支持白名单和开放模式。

### 连接稳定性
Zalo 个人版使用非官方协议，连接可能偶尔断开。如果发生这种情况，让你的 AI 员工重新连接：
> "重新连接 Zalo 个人版"

## 进阶技巧

**语音消息**：发送语音消息——机器人会转录并回复。

**图片与文件**：发送照片或文档，机器人可以分析其内容。

**多语言**：机器人会根据你使用的语言回复——支持越南语、英语、中文等。

**多渠道**：你可以在同一个 AI 员工上同时连接 Zalo（官方）和 Zalo 个人版（非官方），以及 Telegram、WhatsApp 等其他渠道。

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 机器人无响应（官方） | 确认 Bot Token 是否正确，在 OpenMax Dashboard 检查连接状态 |
| 机器人无响应（个人版） | 让你的 AI 员工重新连接——会话可能已过期 |
| 其他人无法与机器人对话 | 默认仅管理员可用，开启白名单或开放模式即可 |
| 账号被警告（仅个人版） | 停止使用非官方集成，切换到 [Zalo（官方）](#zalo-官方-使用方法) |
| 想要断开连接 | 在员工详情页点击对应 Zalo 卡片的**断开连接**按钮，或告诉你的 AI 员工 |
