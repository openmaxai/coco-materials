# 人力资源

AI驱动的招聘、员工管理和HR运营用例。

## 1. 批量简历解析与企业微信意向确认

> 批量解析简历，企业微信自动确认求职意向，筛选效率提升5倍，到场率提升40-45%。

::: details 痛点与解决方案

**痛点：简历筛选耗时巨大，电话确认意向转化率低，面试爽约率高**

招聘旺季，HR团队每天面对数百至数千份简历，人工逐一筛选极其耗时，且简历质量参差不齐，大量不符合岗位要求的简历浪费筛选时间。电话确认面试意向时，接通率往往不足50%，即使确认到场，面试爽约率仍高达30-40%，严重影响招聘效率和业务用人需求的满足速度。

**OpenMax如何解决**

1. **批量简历智能解析与评分**：OpenMax自动解析上传的简历（PDF、Word、图片格式），提取教育背景、工作经历、技能关键词，与岗位JD进行匹配评分，输出优先面试候选人名单。
2. **企业微信自动意向确认**：OpenMax向候选人发送企业微信消息，确认面试意向、可面试时间段、薪资期望，通过更便捷的文字沟通方式替代电话，响应率显著提升。
3. **面试提醒与追踪**：面试前24小时和2小时自动发送提醒，未确认参加的候选人自动标记为需要人工跟进，大幅减少爽约率。

:::

::: details 量化结果与受益角色

**可量化的结果**

- **简历筛选效率**：提升约5倍
- **候选人意向确认时间**：从电话沟通人均15分钟 → 自动化，近乎零人工
- **面试到场率**：提升40-45%

**受益角色**

- **HR招聘专员**：从机械筛选中解放，专注候选人体验和深度评估
- **业务部门**：岗位需求满足速度加快，用人效率提升
- **候选人**：沟通更便捷，面试体验更顺畅

:::

## 2. 一体化AI层：HR招聘+财务发票处理

> 单一AI层覆盖HR简历解析与面试排程、财务发票OCR，月末加班成为历史。

::: details 痛点与解决方案

**痛点：HR与财务的重复性工作各自为战，月末两个部门同时加班，效率低下**

中型企业的HR部门（简历初筛、面试排程、入职资料收集）和财务部门（发票收集、报销审核、账单对账）各自承受大量重复性行政工作，且两个部门的月末高峰往往同时出现，加班成为常态。这些工作高度标准化、规则明确，却占用了大量受过专业训练的HR和财务人员的宝贵时间。

**OpenMax如何解决**

1. **HR端AI自动化**：OpenMax承担简历批量解析、候选人意向沟通、面试时间协调、入职资料收集等标准化HR行政工作，HR专员专注候选人评估和雇主品牌建设。
2. **财务端发票OCR与审核**：OpenMax自动识别员工上传的发票，核查合规性，检测重复报销，生成审核结果，财务仅需处理异常项和最终审批。
3. **跨部门统一工作台**：HR和财务的自动化请求在同一平台管理，行政支持团队可统一监控，避免资源孤岛，月末加班问题从根本上解决。

:::

::: details 量化结果与受益角色

**可量化的结果**

- **HR行政工时**：月度减少约60%
- **财务报销处理时间**：月末高峰处理量不变，人工时间减少约70%
- **月末加班情况**：基本消除

**受益角色**

- **HR团队**：从行政事务中解放，专注核心人才策略
- **财务团队**：月末压力大幅减轻，差错率降低
- **公司管理层**：两个关键支持部门效率同步提升，管理成本降低

:::
