# Discord — Usage Tips

Use your OpenMax AI employee in Discord for developer communities, support servers, and team collaboration.

## Basic Usage

### Direct Messages
Send any message directly to the bot. No @mention needed in DMs — just open a DM with the bot and start typing.

### Server Channels & Threads
@mention the bot in any server channel or thread:
```
@OpenMax AI Employee summarize the latest release notes
```

## Best Use Cases

### Developer & Community Servers
Deploy in developer or community servers for instant, always-on help:
- Answer setup and troubleshooting questions
- Summarize long discussion threads
- Draft announcements and changelogs on demand

### Community Support
In support channels:
- @mention to triage incoming questions
- Translate messages for international members
- Surface docs and answers without leaving Discord

### Team Collaboration
Discord works well for distributed teams and side projects:
- Keep project discussions and tasks in dedicated channels
- Use threads to keep busy channels organized
- Share files and snippets for quick analysis

## Pro Tips

**Smart mode**: Enable smart mode in server channels — the bot receives all messages and decides when to respond based on context, without needing @mentions every time.

**File & image sharing**: Drop images, PDFs, and other files directly in chat. The bot can read, analyze, and respond to attached content.

**Reactions**: The bot can add emoji reactions to acknowledge and respond to messages.

**Threads**: The bot supports replying inside threads, keeping conversations organized in busy channels.

**Owner**: The first person to DM the bot becomes the Owner, who always has full access regardless of access settings.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Bot offline | Verify the Bot Token is correct and the Message Content Intent is enabled in the Discord Developer Portal |
| No reply in a server channel | Ensure the bot is invited to the server, has permission to view and send messages in the channel, and is @mentioned |
| Bot sees empty or no message text | Enable the **Message Content Intent** in the Bot tab of the Discord Developer Portal |
| Cannot send DMs to the bot | DMs work without a server invite — open a direct message with the bot and send any message |
| Want to disconnect | Click the **Disconnect** button on the Discord card in the employee detail page |
